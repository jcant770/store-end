# store-end

## Description 
A backend for a mysql database of an e-commerce site.

## Usage
First enter 'mysql -u root -p' into your terminal
enter your password when prompted
Once there you will need to
* 'source db/schema.sql'
* 'quit'
* 'npm run seed'
* 'npm start'

## Made-with
* Mysql2,
* Express,
* Sequlize,
* dotenv
